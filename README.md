Notes is a privacy oriented, Secure ,beautiful and fast application made in flutter, it supports various features like adding and saving notes. Hiding notes with password security, hidden notes are AES encryoted, Archiving them. To make UI look beautiful it has various UI colors with the support of light and black theme.

<img src="https://i.imgur.com/F3z3OnU.png" height="800">

---

## Features

+ Create and Save Notes
+ Archive them to avoid clutter
+ Hide your Notes
+ Hidden notes are AES encrypted (tied with your password)
+ Multiple UI colors
+ Backup and Restore option
+ Recycle Bin
+ Privacy friendly
+ Without any permission
